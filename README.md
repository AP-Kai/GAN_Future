## 1.2 项目介绍

本项目基于PaddlePaddle和Wechaty开发，功能有：

1. 基于礼貌的打招呼
2. 基于PaddleGAN的人脸融合
3. 基于爬虫的以词搜图
4. 基于云端接口的天气查询
